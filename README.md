## Build
Ensure that you have Python <3.12, >=3.9 installed. Or use a venv

```shell
$ git clone https://github.com/martincarapia/GoProStreamRelay.git
$ cd GoProStreamRelay
$ pip install -r requirements.txt
$ python gui.py
```

And Enjoy!
